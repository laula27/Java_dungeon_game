What the files are:

xmlFiles contains the xml files that define the dungeons that will be used to test your project.

RogueProjectInstructions gives an overview of the entire project.

The MyProject directory contains information about my implementation of the project. These are not project requirements, and you do not need to follow my implementation as long as your project does what it needs to do. Within this directory are the following files:

myGameDescription.pdf give an overview of how I implemented the game.

parserXML.pdf gives a list of classes and packages I used.
